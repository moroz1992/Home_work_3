# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории. Mac0s, Linux
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, Mac0s:
```sh
ls
```

Удаления файла в Windows:
```sh
del <filename>
```
в Linux, Mac0s:
```sh
rm <falename>
```
